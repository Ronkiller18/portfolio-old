📘 Explanation

This lab demonstrates common web vulnerabilities including Stored XSS and Open Redirect.

  • XSS: Occurs when user input is executed as code due to unsafe rendering.
  • Fix: Use safe methods like textContent.
  • Open Redirect: Redirecting users based on untrusted input.
  • Fix: Validate and restrict allowed domains.

📘 Clickjacking Explanation

  • Clickjacking: Tricking users into clicking hidden elements.
  • How it works: Invisible layers are placed above visible UI.
  • Impact: Users perform unintended actions.
  • Fix: Prevent framing and ensure UI integrity.
